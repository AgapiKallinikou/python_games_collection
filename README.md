# Python Games Collection 

This project contains a collection of four games, divided into two categories: games for kids and games for adults.  
The main function `main()` starts the program and guides the user to choose a category and a game.

## Games for Kids
- **Tic-Tac-Toe**  
  The classic two-player game on a 3x3 board. Each player places their symbol (X or O), and the program checks for a win or a tie.

- **Hangman**  
  The user tries to guess a word from a predefined list. They have `n + 4` attempts to guess all the letters. The program shows correct and incorrect guesses after each attempt.

## Games for Adults
- **Roulette (Simplified)**  
  The user inputs the amount of money they have, their bet, and a number between 0–36. If the number matches the computer's random number, the user wins the bet. The player can continue playing until they run out of money.

- **Slot Machine**  
  The user chooses a bet amount and the program displays 4 symbols. If all symbols match, the user wins the bet; otherwise, they lose it. The game continues until the user runs out of money.

## Features
- Replayable: the user can play the same game again or switch to another game without restarting the program.
- Input validation: numbers only for bets, letters only for Hangman, valid choices for games.
- Text-based interface with clear prompts and guidance messages.

## How to Run
To start the program, simply run the function below and follow the on-screen instructions.

```python
main()
```
